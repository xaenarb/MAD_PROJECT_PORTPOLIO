## Lab 4: Book Entry UI & Model
**Focus:** ConstraintLayout and Object-Oriented Programming (OOP).

### Description
The beginning of a library management app ("Shelved"). This lab focuses on creating a precise form for entering book details and creating the backend data structure.

### Key Features
* **ConstraintLayout:** Using Anchors and Guidelines to align labels and input fields perfectly.
* **Java Data Model:** Creation of a `Book.java` class with:
    * Private attributes (encapsulation).
    * Constructors.
    * Getters and Setters.
