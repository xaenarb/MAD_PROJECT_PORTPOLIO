## Lab 3: Calculator
**Focus:** Nested Layouts and Interface Implementation.

### Description
A standard calculator application capable of performing basic arithmetic operations. The UI is designed using nested layouts to create a grid of buttons.

### Key Features
* **Nested LinearLayouts:** Combining horizontal and vertical layouts for a grid structure.
* **Layout Weight:** Using `android:layout_weight` to ensure buttons are evenly distributed across the screen.
* **View.OnClickListener:** Implementing the interface at the class level to handle multiple button clicks within a single method.
