## Lab 5: Multi-Activity Navigation
**Focus:** Intents, Life Cycle, and Advanced Widgets.

### Description
Expansion of the "Shelved" app into a multi-screen application. It includes a secure login page that navigates to the main book entry screen upon successful authentication.

### Key Features
* **Intents:** navigating from `UserLogin` activity to `MainActivity`.
* **Validation:** Regex checks for specific email formats (e.g., `@aiu.edu.my`).
* **DatePickerDialog:** A pop-up calendar widget for selecting dates.
* **State Management:** Handling UI changes (e.g., changing text background color when toggling between "E-Book" and "Regular").
