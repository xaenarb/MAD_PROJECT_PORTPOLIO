## Lab 2: Login Interface
**Focus:** Linear Layouts and Input Configuration.

### Description
A functional login screen user interface designed using XML. It features a logo, specific input fields for credentials, and a submit button, organized vertically.

### Key Features
* **LinearLayout:** Using vertical orientation to stack UI elements.
* **ImageView:** Displaying application assets/logos.
* **Input Types:** Configuring `EditText` for `text` and `numberPassword` to mask sensitive data.
* **UI Linking:** Connecting XML IDs to Java variables using `findViewById`.
