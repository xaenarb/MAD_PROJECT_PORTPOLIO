## Lab 1: User Greeting App
**Focus:** Introduction to Android Studio, Basic UI Components, and Event Handling.

### Description
A simple application that prompts the user to enter their name. Upon clicking a submit button, the app displays a dynamic greeting message using a Toast.

### Key Features
* **TextView & EditText:** Capturing and displaying text.
* **Button & OnClickListener:** Handling user clicks anonymously.
* **Toast:** Displaying transient pop-up messages ("Welcome, [Name]").
